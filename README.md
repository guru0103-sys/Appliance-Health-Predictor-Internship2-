# Appliance-Health-Predictor-Internship2-
📌 Overview

The Predictive “Device Health” Monitor for Household Appliances is an IoT-based monitoring and predictive analytics system designed to track appliance performance in real time and forecast potential failures before they occur.

This project was developed as part of a 6-week Online Internship in IoT Data Analysis at NIELIT Virtual Academy.

The core objective is to shift from reactive maintenance (repair after failure) to predictive maintenance (prevent failure before it happens), improving appliance lifespan, reducing downtime, and enhancing energy efficiency.

🚀 Problem Statement

Household appliances often fail unexpectedly due to:

Overheating

Voltage fluctuations

Irregular usage patterns

Gradual component wear

Traditional maintenance approaches only respond after breakdown, leading to:

Increased repair costs

Unplanned downtime

Energy inefficiency

Reduced equipment lifespan

This project aims to provide a smart, data-driven solution to these challenges.
